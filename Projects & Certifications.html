Movie Ticket Booking System

Description: Developed a Movie Ticket Booking System using MySQL, Tkinter (Python), and user authentication to allow users to browse movies, book tickets, and interact with a database for real-time updates.

MySQL Database Interaction: The system uses mysql.connector to connect to the MySQL database, perform queries, and fetch/update data for movie listings, show timings, user authentication, and ticket bookings.

User Authentication: Login functionality is implemented with a query to validate user credentials (username and password) against the users table, providing a secure login mechanism.

Tkinter GUI: The GUI is created using Tkinter, where users interact with the system to log in, select movies, view shows, and book tickets. The interface includes input fields, buttons, and dropdown menus for user interaction.

Backend Logic for Booking: When booking tickets, the system checks seat availability in the shows table and updates the seat count upon successful booking, while adding the booking details to the bookings table and managing payment status.

Link to the project: Movie Ticket Booking System

Key Features of the Movie Ticket Booking System:

  • Secure user authentication using MySQL for login and password validation.
  • Real-time ticket booking with seat availability checks and booking confirmations.
  • Backend logic to handle movie show timings, bookings, and payment management.
  • Interactive Tkinter GUI for smooth user interaction.
